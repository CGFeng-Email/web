尼龙线

尼龙线，尼龙材质捻合而成，拉伸力强、耐磨耐高温适用于皮革缝纫、垂钓、医疗缝合等多领域。

邦迪线

邦迪线由尼龙、涤纶长丝捻合并粘合处理而成，拉力强、耐磨、缩水小，适用于车座、皮具、安全气囊等缝制。

防水线

防水线采用特殊工艺处理，防水性能优异，耐抗拉适用于户外用品、、雨具、防水服饰等缝制场景。

上蜡线

上蜡线经蜡质浸润处理，顺滑耐磨、防水防潮，适用于皮鞋缝制、皮具修补及户外用品等场景。

透明线

透明线以高透明度尼龙或涤纶制成，隐形美观、韧性强，适用于轻薄面料、饰品装饰及透明材质缝合。
