关于德舍

模块化电力装备与特种集装箱系统服务商
广东德舍电力装备有限公司是一家集设计、生产、销售、安装于一体的特种装备制造企业，总部位于广东省肇庆市金利镇。公司占地面积约8000平方米，年产能达600台特种装备，以模块化技术为核心，致力于为能源基建及工业制造及领域提供场景化、低碳化的空间解决方案。
本公司专注特种集装箱研发、制造与系统集成，以电力预制舱、箱式变电站（箱变）为核心产品，同步拓展储能箱、模块化房屋及集装箱零部件生产。
公司以“让空间承载无限可能”为使命，深耕模块化技术纵深，构建覆盖能源装备-工业组件-人居空间的立体化产品生态，追求完善科学管理，以人才战略为依托，以具有高度社会责任感的企业形象面向社会。
