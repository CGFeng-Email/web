上海纳艾升电器设备有限公司

2017 年，上海纳艾升机电设备有限公司本着“诚信为本 服务至上”的宗旨，获得罗兰数码（Roland）的认可，专业负责罗兰数码（Roland）相关设备的销售和技术支持业务，销售的设备包括喷绘设备、原型建模机、刻字机、3D 打印机、金属雕刻机和写真打印机。通过与罗兰数码的合作，以最具创新性的解决方案以及优秀的售后服务来使客户获得最好的使用体验。

2018 年始，上海纳艾升开始专注于科研和创新领域。为瑞安集团、吉林大学国家新材料实验室等公司与机构提供服务与支持；并且为高校提供教育设备，实验室建设方案等。

2019 年，上海纳艾升与上海同济大学、上海海事大学进行深度合作，开发科创课程，建立完整的科创课程体系；并且与吉林大学，南方科技大学，广州佛山科学院，北京医学院、之宝（中国）户外产品有限公司（ZIPPO）、雀巢公司等高校和企业达成项目上的合作。

2020 年，上海纳艾升与罗兰数码共同开发科研与创新产业链，并成为罗兰（中国）一级代理商，公司已集软件，硬件为一体，并逐渐扩大服务范围。急客户之所急，想客户之所想，帮客户之所需 ，纳艾升为客户提供最优质的产品及服务，并为客户事业的发展与成功提供助力。
