硅麦画锡机
画锡机是一款专门用来进行半导体硅麦产品画锡的设备，左右两侧有上下料机构，进行自动上下料和自动生产，采用封闭式的结构。其主要是用来进行硅麦画锡工艺。集成了鹏创达的优秀的工艺控制功能。根据需求可选择单头、双头、三头画锡。
特点：
  1. 方便快捷视觉定位软件编程，Window操作系统
  2. 精密的流体控制，实现高精度画锡
  3. 自动识别Mark点，自动位置补偿
  4. 高精密丝杆+伺服运动系统，精密三轴运动机构实现快速画锡
  5. 自动上下料，可实现巡视操作系统模式，节约人力资源
  6. 胶路检测功能（缺胶、断胶
